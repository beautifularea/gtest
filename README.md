# gtest
google-test useages

*Build google-test from source 

1. git clone google-test
2. mkdir build && cd build 
3. cmake ..
4. sudo mv libgtest.a /usr/local/lib
5. sudo mv libgtest_main.a /usr/local/lib
6. sudo mv gtest(headers) /usr/local/include

create test files:
1. -----go.h-----
2. ------go.cc----
3. ------go_unittest.cc-----
